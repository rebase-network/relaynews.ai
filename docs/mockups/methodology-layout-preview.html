中立收录

站点通过运营者提交与审核进入目录。被收录并不代表会自动获得靠前排名。

可观测证据

评测排名由各模型分类的实测可用性、延迟、稳定性与性价比信号共同决定。

赞助分离

赞助展示会保持清晰可辨，绝不会改写评测榜单中的实测排序。

可申诉纠偏

如果站点被错误归类，运营者可以提交修正、最新地址或申诉证据供平台复核。

给站点运营者

如果你需要提交资料或发起复核，先确认公开 Base URL、支持模型与测试结果一致。目录优先保留可验证、能复测、能持续观察的信息。

提交站点 先做一次测试

会影响排名

  1. 可用性与连续成功率

  2. 同模型下的延迟与近期一致性

  3. 相对同类站点的价格效率

  4. 稳定性、事故新鲜度与样本置信度

不会改变排名

  1. 赞助展示或其他推广合作

  2. 没有测试证据支撑的人工调位

  3. 无法复现、缺少最新证据的个例反馈

  4. 单次自助测试结果，它只用于诊断

复核路径

  1. 资料、端点或支持模型变化后，重新提交最新信息。

  2. 若公开状态不准确，提供可复现的时间窗口、模型和测试证据。

  3. 补充证据期间，条目可能进入观察或暂停展示，但赞助展示仍与评测排名分离。

建议动作顺序

  1. 先做一次受限测试，确认路由与模型行为。

  2. 再提交 URL、联系方式和必要说明。

  3. 最后持续观察公开榜单、状态变化和备注。
